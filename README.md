A structured repository containing clean programming implementations of the Cambridge International AS & A-Level Computer Science (9618) syllabus, paired with a curated archive of yearly past examination papers (2021–2023).

This project serves as a comprehensive reference for fundamental computer science theory, advanced data structures, algorithmic design, and problem-solving methodologies required for Papers 2 and 4.

🚀 Key Features & Syllabus Coverage
    1. Advanced Practical & Programming (Paper 4)
    Abstract Data Types (ADTs): Complete implementations of core data structures including Linked Lists (singly and doubly linked), Binary Trees, Stacks, Queues, and Hash Tables.

    Algorithms & Efficiency: Standard implementations for search and sort algorithms (Bubble Sort, Insertion Sort, Linear Search, Binary Search) alongside fundamental complexity analysis.

    Object-Oriented Programming (OOP): Demonstrations of key OOP paradigms including encapsulation, inheritance, polymorphism, and custom class construction with robust data validation.

    Low-Level Operations: Practical code handling File Processing (sequential and random access) alongside standard Exception Handling techniques.

    2. Fundamental Problem-Solving & Software Engineering (Paper 2)
        Computational Thinking: Algorithmic solutions utilizing sequence, selection, and iteration constructs.  Data Handling: Implementations involving multi-dimensional arrays, user-defined record types, and string manipulation.Recursion: Classic recursive algorithms mapping out call stacks and termination conditions.
    
    3. Examination Archive (2021 - 2024)
    Categorized repository of official Cambridge examination materials spanning all main variants (Variant 1, 2, and 3).

    Includes Question Papers (QP) paired directly with their official Marking Schemes (MS) for streamlined assessment tracking.
